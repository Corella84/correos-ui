# Correos UI

Proyecto Remix con Shopify Polaris.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

## Rutas

- `/` - Página principal
- `/orders` - Lista de órdenes
- `/orders/:id/review` - Revisar orden
- `/orders/:id/confirm` - Confirmar orden
- `/orders/:id/result` - Resultado de orden
